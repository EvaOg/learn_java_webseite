User Input

In Java, user input can be obtained using the `Scanner` class from the `java.util` package. This class provides methods to read different types of input (such as integers, doubles, and strings) from the keyboard. By creating a `Scanner` object and using its methods, you can capture user input during program execution. Example Exercise: Write a program that prompts the user to enter their name and age, then prints a personalized greeting message along with their age. 

 
 
                 import java.util.Scanner; 
 
  
 
              public class UserInputExample { 
 
              public static void main(String[] args) { 
 
              Scanner scanner = new Scanner(System.in); 
 
  
 
              System.out.print("Enter your name: "); 
 
              String name = scanner.nextLine(); 
 
  
 
              System.out.print("Enter your age: "); 
 
              int age = scanner.nextInt(); 
 
  
 
              System.out.println("Hello, " + name + "! You are " + age + " years old."); 
 
              } 
 
              }

Conditional Statement

In Java, conditional statements such as `if`, `else if`, and `else` allow you to execute different blocks of code based on specified conditions. These statements evaluate boolean expressions and execute corresponding blocks of code if the condition is true. Example Exercise: Write a program that determines whether a given number is positive, negative, or zero, and prints an appropriate message. 

 
 
                 import java.util.Scanner; 

              public class ConditionalStatementsExample { 
 
              public static void main(String[] args) { 
 
              Scanner scanner = new Scanner(System.in); 
 

              System.out.print("Enter a number: "); 
 
              int num = scanner.nextInt();
 

              if (num > 0) {
 
              System.out.println("The number is positive."); 
 
              } else if (num < 0) {
 
              System.out.println("The number is negative."); 
 
              } else { 
 
              System.out.println("The number is zero."); 
 
              } 

              } 

              }

Loops

Loops are essential constructs in programming for repeating specific tasks. In Java, there are different types of loops, but I’ll focus on the for loop. The for loop allows you to execute a code block a fixed number of times. Here’s the syntax for using a for loo 

                
                  for (initialization; condition; update) {
    // Code block to be repeated
}
  • Initialization: You can initialize a variable that acts as a counter for the loop.
  • Condition: The loop runs as long as this condition is true.
  • Update: Here, you can update the counter (e.g., increment or decrement).
  • Let’s look at a simple example. Suppose you want to print numbers from 0 to 4: 
                
                  public class ForLoopExample {
    public static void main(String[] args) {
        for (int i = 0; i < 5; i++) {
            System.out.println(i);
        }
    }
}

Arrays

Arrays are used to store multiple values in a single variable. Instead of declaring separate variables for each value, you can use an array to group them together. Here are the basics of working with arrays in Java: Declaring an Array: To declare an array, define the variable type followed by square brackets. For example 1 

                
                 String[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
                  System.out.println(cars[0]); // Outputs Volvo

Accessing Array Elements: You can access an array element by referring to its index number. Remember that array indexes start with 0. For instance: 2 

                
                 String[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
                  System.out.println(cars[0]); // Outputs Volvo

Changing Array Elements: To modify the value of a specific element, refer to its index and assign a new value: 3 

                
                 cars[0] = "Opel";
                  System.out.println(cars[0]); // Now outputs Opel instead of Volvo
